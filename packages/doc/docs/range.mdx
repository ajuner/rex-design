## 使用场景

滑块控件可以让我们通过在连续或间断的区间内滑动锚点来选择一个合适的数值。

注意事项：

- onChange 是和 value 进行配置做受控处理的。onChange 在滑动过程中不会触发，滑动停止后会触发。
