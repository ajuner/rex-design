## 中文规范  

字体大小在不同的媒介上设计使用的单位会不一样，设计显示屏幕内容时主要使用 PX、DP 单位

我们设计的文字大小，主要通过对应的屏幕上显示时，根据屏幕精度（硬件对焦英寸）和分辨率，在配合屏幕观看距离（对应的物理高度），通过比较科学的公式化计算，再加入最佳实践后的设计微调，沉淀出不同的文字范围以保障信息的可读性。
