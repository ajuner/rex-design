#### 准确

可视化表达要正确传达数据信息，选择正确的图表组件，忠实反映数据里包含的信息。

#### 效率

信息传达有重点，克制不冗余，避免信息过载，必要的标注不能省，不必要的堆叠和信息不要。

用最适量的数据表达对用户最有用的信息。

#### 通俗易懂

图表表达清楚易懂，逻辑条理清楚，可以帮助用户快速达成目标，在最少的时间内获取更多信息。

尽量不要搞三维的图表。

#### 愉悦性

对数据完美表达，合理利用视觉元素进行设计，不过度修饰，给用户优雅的体验。

动效使用需要克制，除非特别的情感化场景（如大屏）。
