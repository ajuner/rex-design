# 设计愿景

![image.png#width=1439px](https://hippo-oss.oss-cn-hangzhou.aliyuncs.com/rex/design-images/values/bc61e589.png)

Rex Design OS 孵化于行业领先的生鲜零售 O2O 变革者，立足于更高效地推进生鲜行业 Farm2Table 的全链条数智化升级改造，**提供给消费者更为确定性更高品质的生鲜消费体验**；为了实现这一点，首先需要最大地释放出生鲜供应链及零售企业内的经营及生产活动的效能。

我们将企业内的组织职能分为经营者，管理协作者，作业者三种核心职能，Rex Design OS 致力于：

- 为经营者提供具有**角色化**的特征的**经营工作台**，能提供**全局多维**的视图，和**指导性**的经营范式，提升**分析和洞察的效率**；另外基于经营者在工作空间的不固定性，提供相应的**移动化**的作业工具。
- 为管理协作者提供真实现场与数字化世界的**准确映射（数字孪生）**，打造**沉浸而聚焦**的**管理工作站**，提升**管理和协作效率**；
- 为作业者提供**简单**，**高效**，**可靠**的**作业工具**及软硬件一体化的解决方案，提升数字化**作业效率**；

# 设计价值观

![大图-最新人物定稿3.30ai-01.jpg#width=5001px](https://hippo-oss.oss-cn-hangzhou.aliyuncs.com/rex/design-images/values/21f383b3.jpg)

基于我们的愿景使命，我们定义了我们的设计价值观为**真实感**，**角色感**，**连接感。**

## 真实感

新零售是对于人-货-场的数字化重构，系统中每一个数字背后有真实的人员，硬件设备，空间设施，每一个数字跳动的背后，有真实的商品被购买，真实的用户评论和反馈。我们的系统就是对于客观世界的**数字化孪生**。

![大图-最新人物定稿3.30ai-03.jpg#width=417px](https://hippo-oss.oss-cn-hangzhou.aliyuncs.com/rex/design-images/values/8ddaad9f.jpg)

- 我们的数字系统需要客观实时地反应真实世界，并对客观世界做面向对象的建模，因建模过程中的抽象导致的细节折叠应该可控可追溯；
- 对于数字系统的操作，可以传导至现实世界，对现实世界产生影响，并将影响的反馈回传。
- 保持客观现实与数字系统的数据的同步，实时，准实时，在数字系统通过对于时间维度的筛选，可以回溯查看客观世界的历史数据切片；

## 角色感

在过往的零售行业，职业化的角色感来自于制服和工具，在数字化的时代，职业化的角色感更多来自于每日工作高频使用的数字化工具界面。

![大图-最新人物定稿3.30ai-02.jpg#width=417px](https://hippo-oss.oss-cn-hangzhou.aliyuncs.com/rex/design-images/values/57ae7574.jpg)

- 数字化的工作界面本身可以表达岗位角色的心智；
- 数字系统可以记录和习得用户角色的使用习惯，提供特征性的角色化定制能力；
- 角色有成长性，系统也应该表达出相匹配的成长性；
- 系统本身可以反向塑造角色的行为和习惯；

## 连接感

在企业的数字化经营中，绝大部分的工作都需要协作。在界面中表达出这种协作的连接关系，使得每一个协作角色能更好地理解自己在整个协作链路中的位置及上下游的影响。

![大图-最新人物定稿3.30ai-04.jpg#width=417px](https://hippo-oss.oss-cn-hangzhou.aliyuncs.com/rex/design-images/values/4f8efc31.jpg)

- 协作的连接对象可以是人与人的连接，也可以是人与硬件设备的连接；人与人的连接的介质通常亦需通过硬件设备；
- 协作连接的对象为个体，则需提供相应的确定性的个体信息，如果协作连接的对象为硬件，则需提供该硬件设备的状态；系统需提供全局的协作连接关系图谱；
- 连接可以是显性化表达的，亦可以是无感的。
