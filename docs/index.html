开始使用

@rexd/core 基于 React 的多端自适应组件库。

danger

ReX Design 组件库目前仍处于 alpha 阶段中，后续可能出现未在文档中列出的 BREAKING CHANGE。请勿在生产环境下使用。

安装#

yarn add @rexd/core

兼容性说明#

  • 组件库兼容 react ^16.8.0 || ^17.0.0
  • 组件库 用到了一些较新的浏览器 API，只兼容较新版本的 PC 端浏览器或手机浏览器
    • chrome: ?
    • safari: ?

使用#

引入样式#

@rexd/core 依赖于 normalize.css，你需要先引入 normalize.css 然后引入 rexd.css：

<head>
<link href="path/to/node_modules/normalize.css/normalize.css" rel="stylesheet" />
<link href="path/to/node_modules/@rexd/core/dist/rexd.css" rel="stylesheet" />
</head>

如果你使用 webpack 且支持引入 CSS 模块，可以在代码入口处引入以下 CSS 文件：

import 'normalize.css';
import '@rexd/core/dist/rexd.css';

使用组件#

import { Button, Toaster } from '@rexd/core';
<Button
type="primary"
onClick={() => {
Dialog.show({ title: '对话框', content: 'hello rexd' });
}}
>
按钮
</Button>;

切换主题与设备#

通过 AppProvider 可以为子节点生效设置主题与设备。

import { AppProvider } from '@rexd/core';
<AppProvider colorMode="dark" device="phone">
{children}
</AppProvider>;

切换主题与设备底层依赖 CSS 变量，你可以通过 container 指定 CSS 变量的注入位置，如不指定则 CSS 变量的生效节点为 :root。

<AppProvider container={div} colorMode="dark" device="phone">
{children}
</AppProvider>
