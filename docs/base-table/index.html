BaseTable 基础表格

使用场景#

通用的基础表格。该组件在开源表格组件 ali-react-table 的基础上进行了样式上的调整，用法详见开源文档。

日常业务推荐使用封装程度更高的 进阶表格组件.

function CaretDown(props) {

    <svg
      focusable="false"
