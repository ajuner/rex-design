Select 选择器

使用场景#

代替原生 select 元素。

使用说明#

  • 单选： <Select multiple={false} /> 或 <SingleSelect />
  • 多选： <Select multiple={true} /> 或 <MultiSelect />
  • 在 TypeScript 项目中推荐使用 SingleSelect/MultiSelect，value 的类型更加明确

示例#

基本示例#

请选择
非内联模式
const basicSelectDataSource = ['盒马', '淘宝', '天猫', { value: '飞猪', label: '飞猪', disabled: true }];


function Basic() {
  const [value, onChange] = useState('');


  return (
    <SingleSelect
      style={{ width: 200 }}
      value={value}
      onChange={onChange}
      hasClear
      hasArrow
      dataSource={basicSelectDataSource}
    />
  );
}


render(<Basic />)

API#

SingleSelect 与 MultiSelect 的区别仅在于 value 的类型不同（defaultValue, onChange 的类型也有相应的区别），其他 props 都相同。
