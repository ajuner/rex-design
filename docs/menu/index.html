Menu 菜单

使用场景#

为页面或区块提供上下文菜单，或是收纳操作元素。

示例#

基本用法#

选项 1
选项 2
Option 2
内联模式
function Basic() {
  return (
    <Menu
      style={{ width: 200 }}
      dataSource={[
        { key: '1', label: '选项 1' },
        { key: '2', label: '选项 2', disabled: true },
        { key: '3', label: 'Option 2' },
      ]}
      onItemClick={(key, detail) => {
        console.log('YOUR click', key, detail);
      }}
    />
  );
}

内联模式
function MultiSelect() {
  const [selectedKeys, onSelect] = useState(['1', '3']);


  return (
    <div>
      <p>selectedKeys: {selectedKeys.join(',')}</p>
      <Menu
        style={{ width: 200 }}
        dataSource={[
          { key: '1', label: '选项 1 ', helper: 'Ctrl+P' },
          { key: '2', label: '选项 2' },
          { key: '3', label: 'Option 3' },
        ]}
        selectMode="multiple"
        selectedKeys={selectedKeys}
        onSelect={onSelect}
      />
    </div>
  );
}

function UncontrolledSingleSelect() {
  return (
    <Menu
      style={{ width: 200 }}
      dataSource={[
        { key: '1', label: '选项 1 ', helper: 'Ctrl+P' },
        { key: '2', label: '选项 2' },
        { key: '3', label: 'Option 3' },
      ]}
      selectMode="single"
      defaultSelectedKeys={['1']}
      onSelect={(nextKeys, detail) => {
        console.log('onSelect', { nextKeys, detail });
      }}
    />
  );
}

function DropdownMenu() {
  // TODO 允许 menu 能够在组件内部直接关闭 popup
  const [visible, setVisible] = useState(false);


  return (
    <Popup
      target={<Button>更多操作</Button>}
      interactionKind="hover"
      hasArrow
      visible={visible}
      onRequestOpen={() => setVisible(true)}
      onRequestClose={() => setVisible(false)}
    >
      <Menu
        dataSource={groupAndSubMenuDataSource}
        onItemClick={(key) => {
          console.log('onItemClick', key);
          setVisible(false);
        }}
      />
    </Popup>
  );
}

API#

字段描述
dataSource*MenuItem[]
菜单项列表
defaultOpenKeysstring[]
默认打开的子菜单
onItemClick(key: string, detail: { item: MenuItem; event: MouseEvent<HTMLDivElement, MouseEvent>; }) => void
菜单项点击回调
classNamestring
styleCSSProperties
子菜单
openKeysstring[]
当前打开的子菜单
onOpen(nextOpenKeys: string[]) => void
子菜单打开或关闭的回调函数
autoCloseSubmenusboolean = true
点击菜单项时，是否自动关闭当前所有已打开的子菜单
interactionKindPopupInteractionKind = 'hover'
打开子菜单的交互方式
选择
selectMode"none" | "multiple" | "single" = 'none'
菜单项选择模式
selectedKeysstring[]
选中的菜单项
onSelect(nextKeys: string[], detail: { item: MenuItem; event: MouseEvent<HTMLDivElement, MouseEvent>; }) => void
选择菜单项的回调
defaultSelectedKeysstring[]
默认选中的菜单项
